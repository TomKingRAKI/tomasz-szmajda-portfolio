TimberKitty

Arcade Game z pełnym stackiem

Data realizacji: Lipiec 2025
Rola: Full-stack Developer
Zagraj online
Powrót do portfolio

O projekcie

An addictive arcade game with a charming twist on the classic Timberman. Built from scratch using pure JavaScript, HTML5 Canvas, and a fully functional Node.js backend for online progress saving.

Kluczowe Cele:

  • Zaprojektowanie gry od podstaw w czystym JS i Canvas.
  • Implementacja backendu Node.js do zarządzania statystykami.
  • Bezpieczne logowanie graczy przez OAuth.
  • Synchronizacja postępów między localStorage a bazą.

Główne Funkcjonalności

Google Login

Integracja OAuth 2.0 z Passport.js dla bezpiecznego logowania.

Online Progress Saving

Zapis statystyk, przedmiotów i osiągnięć w PostgreSQL.

Guest Mode

Zapis w localStorage i automatyczna synchronizacja po logowaniu.

Shop & Equipment

Zakup przedmiotów dających bonusy w grze.

Achievement System

Osiągnięcia za kamienie milowe, zapisywane w bazie.

Dynamic Gameplay

Płynna animacja Canvas z efektami cząsteczek.

Użyte Technologie

Frontend:

JavaScript (ES6+) HTML5 Canvas Tailwind CSS

Backend:

Node.js Express.js PostgreSQL

Inne:

Passport.js (Google OAuth) Netlify Render
