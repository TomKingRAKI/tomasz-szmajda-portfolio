O projekcie

Projekt sklepu Moistry.pl opiera się na WordPress i WooCommerce, dostosowanym od podstaw poprzez utworzenie Child Theme. Wszystkie elementy szablonu zostały zmodyfikowane, aby spełnić wymagania klienta i zachować spójność wizualną.

Całość wdrożenia uwzględniała konfigurację produktów, filtrów AJAX, płatności oraz optymalizację wydajności, gwarantując szybkie i responsywne doświadczenie dla użytkowników.

Kluczowe Cele:

  • Utworzenie Child Theme i dostosowanie szablonu WordPress
  • Konfiguracja WooCommerce z kategoriami i atrybutami produktów
  • Implementacja zaawansowanych filtrów AJAX dla szybkiego wyszukiwania
  • Integracja płatności PayU i Przelewy24 z obsługą webhooków

Główne Funkcjonalności

Child Theme

Dostosowanie motywu Underscores bez modyfikacji oryginalnych plików.

WooCommerce

Tworzenie niestandardowych produktów, atrybutów i ról użytkowników.

Filtry AJAX

Płynne filtrowanie produktów bez przeładowania strony.

Płatności

Integracja PayU i Przelewy24 oraz obsługa powiadomień webhook.

Optymalizacja

Lazy-load, minifikacja CSS/JS, integracja WP Rocket.

Responsywność

Projekt mobile-first i dostępność ARIA roles.

Użyte Technologie

WordPress & WooCommerce:

WP 6.x WooCommerce

Frontend:

HTML5 CSS3/SCSS JavaScript ES6

Inne:

PayU / Przelewy24 API WP Rocket, GitHub Actions MySQL
